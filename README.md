# GWR 
Synthetic Scenes (Lytro Illum 624x432, YUV420) and Realistic Scenes (Canon EOS60D 1024x512, YUV420) are available. If you want to use the dataset for publications, please contact us via yangyou AT hust DOT edu DOT cn, and cite our paper:

Kejun Wu, You Yang, Qiong Liu, Xiao-Ping Zhang, "Gaussian-Wiener Representation and Hierarchical Coding Scheme for Focal Stack Images," in IEEE Transactions on Circuits and Systems for Video Technology, doi: 10.1109/TCSVT.2021.3066523.
